# Java Spring Boot Learning Journey

This repository captures my Java Spring Boot learning journey, with incremental exercises and examples as I work through
Spring Framework fundamentals and Spring Boot basics.

## How to Run the Project
1. Build and run with Maven:
   - `./mvnw spring-boot:run`
   - Or: `mvn spring-boot:run`
2. Open the app at `http://localhost:8080`.

Optional: Build a jar and run it directly:
- `./mvnw clean package`
- `java -jar target/*.jar`
